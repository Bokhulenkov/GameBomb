
# Бомба: игра на скорость и эрудицию

## Описание
"Бомба" — это увлекательное приложение-игра, в котором игроки по очереди отвечают на вопросы из выбранной категории, пока на экране тикает "бомба" с таймером. Задача игроков — дать правильный ответ до того, как бомба взорвется. Если бомба взрывается, игрок получает случайное забавное наказание. Игра идеально подходит для компании друзей, вечеринок или тренировки реакции и эрудиции.
## Особенности
- **Многообразие категорий**: Выбирайте из различных категорий вопросов (например, спорт и хобби, знаменитости, искусство и кино).
- **Таймер**: Таймер для бомбы, который можно менять.
- **Забавные наказания**: Если бомба взрывается, игрок получает случайное наказание.
- **Мультиплеер**: Играйте с друзьями или семьей, создавая незабываемые моменты.
## Технологии
- **Xcode 16**
- **iOS 16.6** и выше 
- Также в разработке использовался пакет **dotLottie Player**      https://github.com/lottiefiles/dotlottie-ios
## Установка
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/Bokhulenkov/GameBomb.git
   ```
2. Откройте проект в Xcode:
   ```bash
   open GameBomb.xcworkspace
   ```
3. Соберите и запустите приложение на симуляторе или устройстве.
## Использование
1. Запустите приложение.
2. Выберите категорию вопросов.
3. Начните игру, отвечая на вопросы по очереди.
4. Следите за таймером и старайтесь ответить до взрыва бомбы.
5. Наслаждайтесь игрой и смейтесь над наказаниями, если бомба взорвалась!
## Примеры вопросов
- Если бы шахматы были олимпийским видом спорта, где проводили бы их финал?
- Что можно делать целый день и не устать?
- Eсли бы знаменитости соревновались в повседневных делах, в чем бы они победили?
## Примеры наказаний
- Три раза хлопнуть в ладоши и сказать: «Я великолепен!»
- Серьезно посмотреть на всех и сказать: «А теперь я исчезаю…» (и спрятаться за ближайшим предметом)
- Изобразить злого кота, который требует внимания
